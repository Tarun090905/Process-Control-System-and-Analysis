# Process-Control-System-and-Analysis

Developed and simulated linear and nonlinear process models using MATLAB to study dynamic system behavior.

Performed stability analysis based on system response and dynamic characteristics.

Applied model identification techniques to derive mathematical models from system data.

Designed and tuned P, PI, and PID controllers in MATLAB, improving closed-loop system performance.

Evaluated system performance using time-domain metrics such as rise time, settling time, overshoot, and steady-state error.

Conducted comparative analysis of open-loop and closed-loop systems, demonstrating enhanced stability with feedback control.
